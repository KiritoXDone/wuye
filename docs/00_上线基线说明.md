# 上线基线说明

## 1. 唯一事实来源

当前仓库以上线口径为准，事实来源优先级如下：

1. `AGENTS.md`
2. `docs/物业管理系统_生产实施与上线交付文档_v1.1.docx`
3. 本目录下的专项补充说明
4. 当前代码实现

## 2. 当前冻结业务口径

- 房间是账单主体，不是账号
- 物业费按面积计费、按年开单、按年缴纳
- 水费按月抄表，录入即出账
- 账单按费用类型分单
- 报表默认按房间统计
- 已支付账单不能直接覆盖

## 3. 当前冻结技术口径

- 后端：Java 17、Spring Boot 3、MyBatis、MySQL 8、Redis、RabbitMQ、Maven
- 管理端：React + Tailwind CSS + TypeScript
- 管理端发布：Nginx 静态托管，需支持快速回滚
- 小程序端：微信小程序 Resident 主链路

## 5. 本次补充能力

### 管理端缴费统计

- 页面路由：`/household-payments`
- 默认按房间口径展示
- 同时显示：
  - 指定年度物业费状态
  - 指定月份水费状态
- 状态支持：
  - `PAID`
  - `ISSUED`
  - `CANCELLED`
  - `MISSING`

### 后台手动标记已缴

- 接口：`POST /api/v1/admin/bills/{billId}/mark-paid`
- 仅允许对 `ISSUED` 账单操作
- 默认按整单全额已缴处理
- 可传入：
  - `paidAt`
  - `remark`
- 会写入 `audit_log`
