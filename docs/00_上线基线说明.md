# 上线基线说明

## 1. 唯一事实来源

当前仓库以上线口径为准，事实来源优先级如下：

1. `CLAUDE.md`
2. `docs/物业管理系统_生产实施与上线交付文档_v1.1.docx`
3. 本目录下的真实对接与补充说明文档
4. 当前代码实现

## 2. 当前冻结业务口径

- 房间是账单主体，不是账号
- 物业费按面积计费、按年开单、按年缴纳
- 水费按月抄表，录入即出账
- 账单按费用类型分单
- 报表默认按房间统计
- 已支付账单禁止直接覆盖

## 3. 当前冻结技术口径

- 后端：Java 17、Spring Boot 3、MyBatis、MySQL 8、Redis、RabbitMQ、Maven
- 管理端目标：React + Tailwind CSS + TypeScript
- 管理端发布：Nginx 静态托管，需支持快速回滚
- 小程序端：微信小程序 Resident 主链路

## 4. 当前仓库状态

已完成收口：

- 年度物业费主开单链路
- 水费录入即出账链路
- 年度物业费支付适配
- 报表、导入导出与年度物业费 / 月度水费口径对齐
- 后端全量测试通过
