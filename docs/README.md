# docs 导航与迁移说明

## 当前事实来源

处理业务、接口、数据库、前端与上线问题时，默认按以下顺序查阅：

1. `../CLAUDE.md`
2. `00_上线基线说明.md`
3. `物业管理系统_生产实施与上线交付文档_v1.1.docx`
4. 本目录下的专项补充文档

## 当前保留文档

- `00_上线基线说明.md`：可 diff 的上线基线摘要
- `05_真实微信登录对接指南.md`：真实微信登录接入说明

- `物业管理系统_生产实施与上线交付文档_v1.1.docx`：当前上线基线原文

## 历史文档说明

仓库此前存在一组“从零开发 / DDL / 联调 / Vue 管理端补充”拆分文档。随着项目收口到上线基线，这些文档已退出默认事实来源：

- 不再作为当前实现依据
- 不应覆盖 `CLAUDE.md` 与上线版文档
- 如需追溯，只能作为历史参考

## 后续维护原则

- 口径变更优先更新 `CLAUDE.md`、`00_上线基线说明.md` 与上线交付文档
- 新增专项说明时，优先放在本目录并保持 UTF-8 编码
- 不再恢复旧的拆分 docs 作为主文档入口
